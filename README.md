# Solenz AI 🧠

Sıfırdan eğitilebilir, kendi kendine öğrenebilen web tabanlı yapay zeka sistemi.

## 🚀 Özellikler
- **Kendi Kendine Öğrenme:** Bilmediği bir soru sorulduğunda internette (DuckDuckGo) araştırma yapar ve öğrenir.
- **NLP Motoru:** TF-IDF ve Cosine Similarity tabanlı özel doğal dil işleme motoru.
- **Eğitim Paneli:** Kendi PDF veya metin dökümanlarınızı yükleyerek AI'yı eğitebilirsiniz.
- **Vercel Hazır:** Serverless fonksiyonlar ve statik site desteği.

## 🛠️ Kurulum

```bash
npm install
npm run dev
```

## 🌐 Canlı Uygulama
Vercel üzerinde yayına almak için bu repoyu Vercel'e bağlamanız yeterlidir.
